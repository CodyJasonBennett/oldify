## 𝖀𝖘𝖆𝖌𝖊

ℭ𝔬𝔪𝔪𝔞𝔫𝔡 𝔏𝔦𝔫𝔢

npm install -g oldify
oldify src dist --bold
```

const oldify = require('oldify');

// 𝕿𝖗𝖆𝖓𝖘𝖕𝖎𝖑𝖊𝖘 𝖋𝖎𝖑𝖊𝖘 𝖆𝖓𝖉 𝖙𝖍𝖊𝖎𝖗 𝖉𝖎𝖗𝖊𝖈𝖙𝖔𝖗𝖎𝖊𝖘

```
